# Masonry Reels shot-sheet

Editable, visual shot-sheet for Masonry short-form reels, served via GitHub Pages.

- `index.html` renders `data.json` into a rich, editable shot-sheet (per-beat reference frames + script). Edit any text on the page and click Save to commit back here (needs a fine-grained GitHub token with Contents: write on this repo).
- `frames/` holds the inspiration reference frames.
- `brand-reference.html` is the standing D2C / celebrity brand catalog.
